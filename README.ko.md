---

**쌈지**는 귀한 것을 넣어 다니던 작은 주머니입니다. 복사한 모든 것을 담아두고, 밀리초 단위로 검색하고, 키 하나로 다시 붙여넣습니다. 모든 데이터는 Mac 안에만 있습니다 — 계정도, 네트워크도, 수집도 없습니다.

## 왜 만들었나

macOS 26(Tahoe)의 클립보드 프라이버시 정책으로 오래 업데이트가 멈춘 클립보드 매니저들이 깨졌습니다. 언제 올지 모를 수정을 기다리는 대신, 새 규칙에 맞춰 처음부터 만들었습니다 — 우회하지 않고 정식 권한으로 시스템과 함께 동작합니다.

## 기능

- **중앙 팔레트** (`⌘⇧V`, 변경 가능) — 검색·리스트·프리뷰가 한 패널에. 앱을 가리지 않는 비활성 패널이라 작업 중인 앱의 포커스를 뺏지 않습니다.
- **다이렉트 붙여넣기** — `Enter`로 현재 앱에 바로 붙여넣기(손쉬운 사용 권한). `Shift Enter`는 복사만, `⌘1–9`는 즉시 붙여넣기.
- **보드** — 클립을 모아두는 고정 공간. 보드는 히스토리와 독립적이라 전체에서 지워도 보드 항목은 남습니다. 생성(`⌘N`), 담기(`⌘P`), 이동(`⌘[` / `⌘]`), 순서 변경(`⌘⇧←→`).
- **시크릿 보드** — 내용 마스킹, 라벨 표시, `⌥` 홀드로만 잠깐 보기. (Touch ID 금고화는 로드맵에 있습니다.)
- **페이스트 스택** — `⌘K`로 여러 개 담고 `⌘⏎`로 한 번에: 줄바꿈/공백/콤마/`&&`/순차 붙여넣기.
- **변환 붙여넣기** (`⌘T`) — 대/소문자, 공백 정리, kebab/snake 케이스, JSON 프리티, 터미널용 변환.
- **진짜 전문 검색** — SQLite FTS5 trigram 토크나이저로 한국어도 부분 문자열 매칭. 디바운스로 타이핑이 절대 밀리지 않습니다.
- **Paste에서 이주** — 보드·라벨·이미지까지 Paste 라이브러리 전체를 클릭 한 번에 가져옵니다.
- **은신 모드** (`⌘⇧E`) — 수집 즉시 중단, 메뉴바 아이콘이 흐려집니다.
- **앱 수집 제외** (`⌘E`) — 비밀번호 관리자 등 원하는 앱은 수집하지 않습니다.
- **보관 정책** — 1~90일 또는 무제한. 보드에 담긴 항목은 기간과 무관하게 보존.
- **다국어** — 한국어·영어, 시스템 언어를 따릅니다.

팔레트에서 `⌘/`를 누르면 전체 단축키 사전이 열립니다.

## 설치

**macOS 15.4+** 필요. 소스 빌드(Xcode 커맨드라인 도구 + Swift 6):

```bash
git clone https://github.com/Minzino/ssamji.git
cd ssamji
./scripts/bundle.sh   # 빌드 + 서명 + /Applications 설치 + 재시작
```

첫 실행 시 권한 두 개를 허용해주세요:

1. **클립보드 접근** — 시스템 설정 → 개인정보 보호 및 보안에서 쌈지를 *항상 허용*으로 (macOS 26).
2. **손쉬운 사용** — 다이렉트 붙여넣기(`⌘V` 합성)에 필요합니다. 없으면 `Enter`가 복사만 합니다.

> 동봉 스크립트는 로컬 자체 서명 인증서를 사용합니다. 공증된 바이너리 릴리스를 준비 중입니다.

## 프라이버시

- 모든 데이터는 `~/Library/Application Support/Ssamji/`에만 저장됩니다.
- 네트워크 접근·분석·계정이 없습니다.
- 시스템이 민감하다고 표시한 내용(`org.nspasteboard.ConcealedType`, 비밀번호 관리자 등)은 수집하지 않습니다.

## 성능

쌈지는 "렉 금지 계약" 위에서 만들어졌습니다: 프리뷰 사전 계산, 행 메모이제이션, CJK 폰트 폴백 사전 해석, 키 반복 중 프리뷰 보류. Apple silicon에서 수백 개 스크롤·보드 전환이 한 프레임 예산 안에 듭니다.
